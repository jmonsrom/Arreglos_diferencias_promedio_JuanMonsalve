Proyecto educativo para ver arreglos en Java

Objetivo del proyecto

Desarrollar un programa en Java que permita almacenar números en dos arreglos, calcular la diferencia entre ellos y mostrar el promedio de los datos ingresados.

Descripción

El programa solicita al usuario ingresar 7 números para dos arreglos. Posteriormente calcula la diferencia entre los elementos correspondientes de cada arreglo y almacena el resultado en un tercer arreglo.

Finalmente muestra:

- Los valores del tercer arreglo
- El promedio de todos los datos ingresados

Instrucciones de ejecución

1. Clonar el repositorio
2. Abrir el proyecto en un IDE o terminal
3. Compilar el programa
